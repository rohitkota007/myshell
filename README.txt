                                                                                    SHELL ASSIGNMENT
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
To compile run  'make'
To execute run  './shell'
Run 'make clean' to remove executables and .o files
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Implementation :-

Reads commands in a loop
Shows prompt
Runs many commands
Runs commands in Background
I/O stream redirection (combination is also implemented)
single-level pipe is implemented
Combination of redirection and pipe is also implemented
Handled Signals like Ctrl-C, Ctrl-D, Ctrl-\ and Ctrl-Z  
"exit" command to exit the shell where it is currently running
Implemented Background processes ('&' operator) // Used Set in C++ Standard Template Library (STL) only for implementing background processes
Foreground processes is under construction
Implemented run command,‘run’ takes one argument which is a file name containing a list of commands separated by newline and then executes the commands one by one. (working partially but still under construction)
